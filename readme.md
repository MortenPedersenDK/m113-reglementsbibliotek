# Pansermuseets PMV M113 reglementer

Dette repository er et website som indeholder reglementer til PMV M113 familien som er udstillet på Pansermuseet i Oksbøl.

Formålet er at gøre reglementet nemmere at benytte som opslagsværk, idet de tidligere indscannede reglementer ikke var søgbare da hver side i princippet var et foto.

I dette repository er der foretaget tekstgenkendelse på alle sider, og ud fra dette er der opbygget en indholdsfortegnelse og et søgeindex, således at når der søges på ord, som indgår i reglementet, så vises en oversigt over sider hvor ordet indgår.

Sitet kan ses [her](https://mortenpedersendk.github.io/m113-reglementsbibliotek).
